# 模版编译

- 正则
  - 开发中的一项单独技能，如写sql
  - 在字符串中，长文中，匹配想要的内容
  - 编写正则表达式
    - 一个字符 1
    - [0-9] 0-9 范围内的一个字符
    - {10} 一个字符的匹配次数
    - ^ 开头匹配
    - $ 匹配的结尾
    - + 贪婪匹配 一次或多次
    - * 0次或多次

- 百度面试题，了解面试官的内心思考，为何出此题？
  - 正则 + replace
  - 百度的当家热门技术 正则 和git 一样重要
  - 正则的细节 
  - 第一种解法：while + replace 介绍基本逻辑
  - 第二种解法：/g + replace 回调函数高级用法
  - 最终进阶：使用递归
